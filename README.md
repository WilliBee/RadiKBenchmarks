# Additional Benchmarks for the RadiK.jl package

**Instatiate environment :**

```julia
using Pkg
Pkg.instantiate()
```

**Add backend :**

```julia
using Pkg
Pkg.add("Metal")
```

or 

```julia
using Pkg
Pkg.add("CUDA")
```

**Running benchmarks :**
```julia
using Metal
backend=MetalBackend()
include("radik_single_task_benchmark.jl")
include("radik_multi_task_benchmark.jl")
```
